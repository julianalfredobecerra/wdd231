WDD 231 - Project Site Plan - Julian Becerra

Site Name

Golden Treasures Jewelry
This name suggests premium, handcrafted pieces and conveys the idea of unearthing something precious.
Optional domain availability: goldentreasuresjewelry.com

Site Purpose

The website will feature a beautiful Homepage, a detailed Catalog page showcasing each jewelry piece with images and descriptions, and a Contact Us page so customers can inquire about custom orders or store locations.

Scenarios

  • How do I browse necklaces by price or material?
  • Where can I find the store's contact information for a custom design request?

Color Schema

  • Accent Gold: #d4af37 - Used for headings, borders, and button hover states.
  • Ivory Background: #f4f1ea - Used for page backgrounds and section backgrounds.
  • Text Dark: #333333 - Used for all body text and captions.

Typography

  • Playfair Display - Used for all headings (h1, h2) to convey elegance.
  • Roboto - Used for paragraphs (p) and lists (ul) for readability.
